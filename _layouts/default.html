Instruções

Como utilizar o sistema de registro

Passo a Passo

1. Selecione o Local

Escolha entre "Silo (Produção)" ou "Armazém" dependendo de onde o carregamento foi realizado.

2. Selecione a Transportadora

Escolha a transportadora responsável pelo carregamento na lista de opções.

3. Preencha os Dados

Informe todos os dados solicitados: ordem de carregamento, motorista, placa, peso e checklist.

4. Envie os Dados

Clique em "Enviar" para copiar a mensagem, "Copiar" para copiar a mensagem ou "WhatsApp" para compartilhar diretamente.

Dica Importante

Após o envio, o sistema automaticamente copiará a mensagem e você pode compartilhar manualmente.
